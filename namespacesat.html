◆ clause_like

template<typename T >
concept sat::clause_like = concepts::typed_range<T, Literal>

Concept modeling a clause. A clause is some range of literals, i.e. it must have accessible members begin() and end() that return an iterator to the first or pas the end elements respectively. For example std::vector<Literal> satisfies this concept.

◆ heuristic

template<typename H >
concept sat::heuristic = concepts::callable_r<H, Variable, const std::vector<TruthValue>, std::size_t>

Concept modelling the heuristic interface. A heuristic is a type that can be called with a vector of truth values (the model) and an unsigned integer representing the number of variables that are undecided and returns a sat::Variable
